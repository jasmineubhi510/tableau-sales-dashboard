# Tableau Sales Dashboard

## Overview
This project presents an interactive Tableau dashboard analyzing sales performance, trends, and key business metrics.

## Dashboard Preview
![Dashboard](sales-dashboard.png)

## Features
- Sales trends over time
- Category performance analysis
- KPI metrics (Profit, Orders, Sales)
- Geographic sales distribution

## Tools Used
- Tableau

## Insights
- Sales increase toward the end of the year
- Technology and key categories drive revenue
- Certain regions show higher sales concentration

## Purpose
This project demonstrates data visualization skills and the ability to communicate business insights effectively.
